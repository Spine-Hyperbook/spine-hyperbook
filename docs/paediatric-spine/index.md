# Paediatric Spine

This is a hub page, not a separate set of conditions. Paediatric spinal pathology is written up on the pages where it clinically belongs (Deformity, Trauma, Infection), and linked from here so someone approaching the topic "by age" rather than "by pathology" can still find everything in one place.

**Trauma**

- Paediatric spinal trauma has distinct injury patterns and clearance criteria from adult trauma, page pending under [Trauma](../trauma/index.md).

**Infection**

- Paediatric discitis has a different typical presentation (younger children, refusal to weight-bear) to adult pyogenic infection, worth its own short section on the [Pyogenic Spinal Infection](../infection/pyogenic-infection.md) page once written rather than a separate page here.

If a topic is genuinely paediatric-specific throughout (no meaningful adult equivalent), it can live directly under this section instead, use judgement rather than forcing everything into the adult pages.
