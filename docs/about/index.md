# About & Contributors

## Why this exists

Foot and Ankle trainees have had BOFAS's Lectures of Distinction and the Hyperbook for years. Spine surgery, split awkwardly across T&O and Neurosurgery training, has had nothing equivalent. This is an attempt to build one, starting with T&O content and adding a neurosurgical layer as the project grows.

## Editorial approach

- Every clinical page is mapped to a specific curriculum reference.
- Every page carries an evidence table citing named studies, not unreferenced claims.
- Where the evidence is genuinely mixed, the page says so explicitly rather than presenting false certainty.
- Content follows standard copyright practice: ideas and findings are summarised and attributed, direct quotation is minimal.

4. All clinical content is reviewed before publication.
